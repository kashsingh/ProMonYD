PromonYD

It is youtube-dl based GUI downloader for downloading video and audio from YouTube and few other sites.

About the program.

PromonYD is a small GUI youtube-dl based program to download videos from YouTube.com and a few more sites. It requires the Python interpreter, version 2.6, and PyQt4 for the interface. It should work in your Unix box, in Windows or in Mac OS X. It is released to the public domain, which means you can modify it, redistribute it or use it however you like. The project is currently being developed at github.

Features

The project is still in beta phase and we are continuously trying to improve it every day. Some of the features include:

  • A large no of site support, it supports a little over 430 sites. For more information visit this page.
  • High speed video/audio download.
  • Download entire YouTube playlist in the best video quality available.
  • Resuming file download.

Upcoming Features

  • Download from batch file or text file which contains URL of videos.
  • Subtitle download.
  • Multiple video/audio download at a given time.
  • Proxy connection information support.

Develop and Bugs Report

If you want to contribute to the project, your assistance is most welcomed here. If you want to create a build of PromonYD yourself, you'll need

  • python 2.7
  • PyQt 4

Bugs and suggestions should be reported at: https://github.com/kashsingh/ProMonYD/issues. Unless you were prompted so or there is another pertinent reason (e.g. GitHub fails to accept the bug report), please do not send bug reports via personal email. Also please elaborate on what feature you are requesting, or what bug you want to be fixed. Make sure that it's obvious

  • What the problem is
  • How it could be fixed
  • How your proposed solution would look like
